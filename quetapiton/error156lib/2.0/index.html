About

A ultra-lightweight, header-only C library for creating Windows GUIs using native WinAPI. Simple, fast, and dependency-free.

Quick Start

Define the implementation in your main C file:

#define ERROR156_IMPLEMENTATION
#include "error156gui.h"

int main() {
    e156_Window app = e156_Init("My App", 400, 300);
    e156_Label(&app, "Hello World!", 20, 20, 100, 20);
    e156_Button(&app, "Click Me", 20, 50, 80, 30, 1);

    while(e156_Loop()) { Sleep(10); }
    return 0;
}

Functions Reference

FunctionDescription
